## 백신도우미 테이블 설계

필요 테이블 3개 

1. 센터 정보 테이블
2. 백신 접종자 수 테이블 
3. 예약자 정보 테이블 

##


# 1. 센터정보 테이블 

 * 테이블 이름 : CENTER_LIST
 * 내용 
    - 일련번호, 센터명, 시설명, 우편번호, 간략주소, 상세주소, 전화번호  
 
 * 제약
    - primary key - center_no -> 자동생성 
    - not null : 전체 
   
   
# 2. 백신 접종자 수 테이블 

 * 테이블 이름 : VACCINE_COUNT
 * 내용 
    - 일련번호, 기준일자, 지역, 1차 접종수, 2차 접종수, 누적 1차 접종수, 누적 2차 접종수

 * 제약 
    - primary key - 일련번호 -> 자동생성 
    - not null : 전체
   
   
# 3. 예약자 정보 테이블 
   
   * 테이블 이름 : RESERVE
   * 내용 
     - 이름, 전화번호, 주소, 주민번호, 접종한 백신명, 1차 접종일
   * 제약
    - primary key - 주민번호 
    - not null : 전체
